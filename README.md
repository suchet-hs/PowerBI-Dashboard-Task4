# 📊 Task 4: Dashboard Design (Power BI)

## 📌 Objective
To design an interactive dashboard for business stakeholders to analyze sales performance.

---

## 🛠️ Tools Used
- Power BI

---

## 📂 Dataset
Superstore Sales Dataset

---

## 📊 Dashboard Features

### 🔹 Key KPIs
- Total Sales: 2.23M
- Total Profit: 283K
- Profit Margin: 12.7%
- Total Orders: 4898

### 🔹 Visualizations
- Sales by Year (Line Chart)
- Sales by Category (Bar Chart)
- Sales by Segment (Pie Chart)

### 🔹 Filters (Slicers)
- Region
- Category
- Order Date
